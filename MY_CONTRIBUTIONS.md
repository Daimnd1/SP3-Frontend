# My Contributions - Deskberg Smart Desk Frontend

This repository contains the frontend for Deskberg, a third-semester team project at the University of Southern Denmark. The system connects a React web application, backend APIs, Supabase services, LINAK desk hardware, and Raspberry Pi Pico W notification workflows.

## Role

Frontend Owner / Full-Stack Developer

I was responsible for the frontend experience and much of the application/business logic that made the distributed system usable from the web interface.

## Main Contributions

- Designed the complete frontend experience in Figma.
- Built the React/Vite frontend structure and main application flow.
- Implemented navigation, authentication flow, dashboard, desk control, configuration, reports, profile, and management views.
- Built business logic for desk connection state, live height polling, posture mode detection, timer tracking, reminders, user settings, and usage analytics.
- Integrated frontend views with backend APIs for desk state and desk control.
- Integrated Supabase authentication and database-related frontend workflows.
- Connected frontend workflows with embedded Pico W notification behavior through the backend/embedded system integration.
- Created reporting and visualization features for posture habits, session duration, sitting/standing balance, and management insights.
- Added Playwright end-to-end tests for core user flows.
- Deployed the frontend through Netlify from the main GitHub branch.
- Collaborated with backend and embedded teammates to connect React, Express.js, Supabase/PostgreSQL, Docker-deployed services, and Raspberry Pi Pico W hardware.
